# Koreanify

## Features

Experience smoother Korean typing with this client-side mod. It effectively resolves IME issues, building upon the enhancements introduced in 26.1-snapshot-9.

26.1-snapshot-9의 개선된 IME를 기반으로 게임 내 한글 채팅 호환성과 입력 환경을 향상시킨 클라이언트 유틸리티 모드입니다.
